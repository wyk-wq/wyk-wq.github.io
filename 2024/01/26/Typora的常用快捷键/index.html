Typora的常用快捷键

发表于2024-01-26|更新于2024-01-26
|阅读量:

¶Typora 常用快捷键：

  1. 切换编辑语法模式/预览模式： Ctrl + /

  2. 标题：

    • 一级标题：# + 空格 + 标题内容
    • 二级标题：## + 空格 + 标题内容
    • …
    • 六级标题：###### + 空格 + 标题内容

  3. 粗体：Ctrl + B

  4. 斜体：Ctrl + I

  5. 表格：Ctrl + T

  6. 代码块：``` + Enter

  7. 引用：> + 空格

  8. 转义字符：\ + 字符

  9. 无序列表：- + 空格 或 * + 空格

  10. 有序列表：数字 + . + 空格

  11. 生成目录：[TOC] + Enter

  12. 选中一整行：Ctrl + L

  13. 选中单词：Ctrl + D

  14. 选中相同格式的文字：Ctrl + E

  15. 跳转到文章开头：Ctrl + Home

  16. 跳转到文章结尾：Ctrl + End

  17. 搜索：Ctrl + F

  18. 替换：Ctrl + H

  19. 插入链接：Ctrl + K

  20. 数学公式：$ + Enter

  21. 表情：: + 单词 + :

  22. 插入图片：直接拖动到指定位置或 Ctrl + Shift + I

¶Typora 常用格式及操作：

  1. 删除线：Alt + Shift + 5 或 ~~内容~~
  2. 下标：<sub>内容</sub>
  3. 上标：<sup>内容</sup>
  4. 脚注：[+^+脚注]
  5. 高亮：==内容==
  6. 着重：Ctrl + Shift + \
  7. 淡化：+内容+
  8. 转义：\ + 符号
  9. 清除格式：Ctrl + \
  10. 居中：<center> + 居中内容 + <center>
  11. 右对齐：<p align=“right”>右对齐</p>
  12. 左对齐：<p align=“left”>左对齐</p>
  13. 换行：回车或 <br/>
  14. 空格（一个汉字俩空格）：  (一个空格) 或   (两个空格）

¶Typora 常用操作：

  1. 新建：Ctrl + N
  2. 新建窗口：Ctrl + Shift + N
  3. 打开：Ctrl + O
  4. 保存：Ctrl + S
  5. 另存为：Ctrl + Shift + S
  6. 关闭：Ctrl + W
  7. 撤销：Ctrl + Z
  8. 重做：Ctrl + Y
  9. 复制为 MarkDown：Ctrl + Shift + C
  10. 粘贴为纯文本：Ctrl + Shift + V
  11. 选中当前行/句：Ctrl + L
  12. 选中当前格式文本：Ctrl + E
  13. 选中当前词：Ctrl + D
  14. 跳转到文首：Ctrl + Home
  15. 跳转到文末：Ctrl + End
  16. 查找：Ctrl + F
  17. 替换：Ctrl + H
  18. 大纲视图：Ctrl + Shift + 1
  19. 文档列表视图：Ctrl + Shift + 2
  20. 文件树视图：Ctrl + Shift + 3
  21. 显示/隐藏侧边栏：Ctrl + Shift + L
  22. 源代码模式：Ctrl + /
  23. 专注模式：F8
  24. 打字机模式：F9
  25. 切换全屏：F11
  26. 实际大小：Ctrl + Shift + 0
  27. 放大/缩小：Ctrl + Shift + + / Ctrl + Shift + -
  28. 应用内窗口切换：Ctrl + Tab
  29. 打开 DevTools：Shift + F12

¶Typora 导出文件：

  1. 文件 → 导出 → 选择要导出文件的格式

请根据需要整理和使用上述快捷键和操作。希望这份整理对您有帮助！
